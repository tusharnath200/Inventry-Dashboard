#Inventory Dashboard Project
 # Overview

The Inventory Dashboard is a data visualization project designed to provide actionable insights into inventory performance, sales trends, and product-level analytics. It enables users to monitor key business metrics such as inventory value, revenue, stock levels, and product performance in an interactive and visually appealing format.

This dashboard helps businesses make informed decisions regarding stock management, sales optimization, and demand forecasting. 
#  🚀 Key Features
📦 Total Inventory Units
Displays the total number of items currently in stock.
💰 Inventory Value
Shows the total monetary value of inventory.
📈 Revenue Tracking
Provides a high-level overview of total revenue generated.
📊 Sales Revenue by Quarter
Visual representation of quarterly sales trends to identify growth patterns.
📉 Revenue Growth Trend
Line/area chart highlighting cumulative revenue progression over time.
📋 Product-Level Details Table
Includes:
Product ID
Product Name
Inventory Units
Selling Price
Inventory Value
🏆 Top 10 Inventory Products
Identifies products with the highest stock levels.
⚠️ Top 10 Lowest Inventory Products
Highlights low-stock items for restocking decisions.
🔍 Brand Filter
Allows filtering data by brand (e.g., Dior) for focused analysis.
